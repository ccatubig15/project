Jquery Raffle with PHP Array


Simple random number generation in a loop to simulate a raffle draw.
The code uses bootstrap as the main design and to produce modal to show
who is the winner. Also it requires jquery to run.

You may use Jquery CDN at
https://cdnjs.cloudflare.com/ajax/libs/jquery/3.3.1/jquery.min.js

and

Bootstrap CDN at
https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/css/bootstrap.min.css
https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/js/bootstrap.min.js
